Command: raidSetup(raidName, day, date, time, timezone)

1. raidName = Name of the raid for which you're trying to setup.
2. day = the day the raid is set for.
3. date = the date the raid is set for.
4. time = the time the raid is set for.
5. timezone = the timezone the raid is set for.

Setup a raidSetup message inside the channel, deleting previous message.

Example:
'~raidSetup trainingRaid Sunday 24/12 8pm CEST'
