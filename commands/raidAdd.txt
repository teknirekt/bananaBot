Command: raidAdd

@param (args[0] = raidName) STRING
@param (args[1] = roleIndex) INT
@param (args[2] = roleDescription) STRING
@param (args[3] = discordName) STRING (officersOnly, optional)

Add author of message/discordName to the specified spot at raidSignup.
Intended for single spot signups. If you want intervals, check out raidFill.

Example:
'~raidAdd trainingRaid 7 CondiSoulbeast'
